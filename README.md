## Description

  This command line cms is a prototype CMS designed to interface with a mysql database and allow the user to input, view, and update database entries for employees, departments, and roles within a company. A user will simply need to follow command line prompts to work with the database. This application uses express, inquirer, and mysql. Big thanks to those who put these packages out there to use for free.
